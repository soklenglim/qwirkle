# Qwirkle

### To compile the code:
```g++ -Wall -Werror -std=c++14 -g -O -o qwirkle Tile.cpp Board.cpp Engine.cpp Player.cpp Node.cpp LinkedList.cpp Menu.cpp qwirkle.cpp```

### To compile using Makefile:
  ```make```
  
### To run the code:
  ```./qwirkle```
  
### To clean up
```make clean```
